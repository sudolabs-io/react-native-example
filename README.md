# React Native reusability example
## Getting started
1. Install node v12
2. Install Yarn `npm install -g yarn`
3. Clone and go to project folder
4. Install dependencies `yarn`
5. Run:
- web: `yarn web`
- android: `yarn android`
- ios: `yarn ios`
